# Amidaments (Terra Studio)

Una aplicació web moderna per a la gestió d'amidaments i pressupostos en el sector de l'arquitectura i la construcció.

## Característiques

- **Gestió de Capítols i Partides**: Organització jeràrquica de projectes.
- **Editor d'Amidaments**: Introducció detallada de quantitats (línies d'amidament).
- **Importació/Exportació BC3**: Suport total per al format FIEBDC-3 (.bc3).
- **Càlcul de PEM**: Gestió de preus desglossats i ajustos globals de pressupost.
- **Exportació PDF**: Generació de pressupostos i resums d'amidaments amb una estètica cuidada.
- **Base de Preus**: Consulta i gestió de recursos i preus unitaris.

## Tecnologies utilitzades

- **React**: Biblioteca per a la interfície d'usuari.
- **Vite**: Eina de construcció ràpida.
- **Tailwind CSS**: Estils moderns i personalitzats.
- **jsPDF**: Generació de documents PDF.
- **SheetJS (xlsx)**: Processament de dades Excel si cal.
- **Lucide React**: Iconografia moderna.

## Instal·lació i Desenvolupament

Per començar a treballar en el projecte localment:

1. Instal·la les dependències:
   ```bash
   npm install
   ```

2. Executa el servidor de desenvolupament:
   ```bash
   npm run dev
   ```

3. Per construir la versió de producció:
   ```bash
   npm run build
   ```

## Disseny (Terra Studio)

L'aplicació segueix una estètica inspirada en la natura ("Terra Studio"), amb tons neutres, terracotes i verds savis, enfocada a arquitectes moderns que valoren la sostenibilitat i el minimalisme.
